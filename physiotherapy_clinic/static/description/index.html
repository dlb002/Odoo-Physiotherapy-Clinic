Gestión de Clínica de Fisioterapia

Este módulo permite gestionar todos los aspectos de una clínica de fisioterapia, incluyendo:

  • Registro de pacientes.
  • Gestión de citas de pacientes.
  • Seguimiento de tratamientos realizados por fisioterapeutas.
  • Visitas de seguimiento.

El módulo incluye funcionalidades avanzadas como la validación de citas para evitar solapamientos, generación de reportes de citas y un sistema de permisos para asegurar que los usuarios solo puedan acceder a la información adecuada.

Características clave

  • Vista de calendario para citas.
  • Control de tratamientos y seguimiento.
  • Permisos personalizados para usuarios.
  • Reportes QWeb para citas en PDF.

Requisitos

Este módulo requiere Odoo versión 15 o superior.

Cómo usarlo

Para empezar a usar el módulo, simplemente instálalo desde la interfaz de aplicaciones de Odoo y configura los parámetros iniciales según las necesidades de la clínica.
